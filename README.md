FilmAraSwiftUI – SwiftUI Film Arama Uygulaması
SwiftUI ile geliştirilmiş, OMDb API kullanan modern bir film arama uygulaması.
Kullanıcı film adını yazar, arama yapılır ve detay ekranından film bilgileri görüntülenir.

Kullanılan Teknolojiler
SwiftUI
MVVM
OMDb API
URLSession
Custom Image Loader (OzelImage)
NavigationView + NavigationLink
DisclosureGroup
